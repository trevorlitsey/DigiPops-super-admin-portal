# DigiPops SuperAdmin Portal

## Getting Started

### Console 1
```
$ git clone https://github.com/trevorlitsey/DigiPops-super-admin-portal.git
$ cd test-admin/server
$ yarn install
$ yarn start
```

### Console 2
```
$ cd test-admin/client
$ yarn install
$ yarn start

--> client server running on localhost:3000
```
